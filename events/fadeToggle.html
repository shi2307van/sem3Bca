jQuery fadeToggle() effect *******************

Description:
--------------

jQuery fadeToggle() method used to toggle between the fadeIn() and fadeOut() methods .If the elements are faded in ,it will make them faded out and if they are faded out it will make them faded in.

Syntax:
---------

$(selector).fadeToggle(); $(selector).fadeToggle(speed); $(selector).fadeToggle(speed,callback); $(selector).fadeToggle(speed,easing,callback);

Parameters

|=====================================

Description

======================================|

Speed :


|======================================

it is an optional Parameters and specify the speed of the delay.it possible value are slow,fast,miliseconds.

======================================|

Easing :


|======================================

It specifies the easing function to be used for transition.

======================================|

Callback :


|======================================

It is an Optional parameter and specify the function to be called after comletion of fadeToggle() effect.

======================================|
