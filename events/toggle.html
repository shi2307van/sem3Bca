jQuery Toggle() effect **********************

Description: ------------

jQuery Toggle() is special type of method which is used to toggle between the hide() and show() method.It show the hidden element and hides the shown element.

Syntax: -------

$(selector).Toggle(speed,callback);

Parameters

|=====================================

Description

======================================|

Speed :

|=====================================

it is an optional Parameters and specify the speed of the delay.it possible values are slow,fast and milliseconds.

======================================|

Callback :

|=====================================

It is an Optional parameter and specify the function to be called after comletion of Toggle() effect.

======================================|

Example:

$("p").Toggle();
