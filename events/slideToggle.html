jQuery slideToggle() effect *******************

Description:
--------------

jQuery slideToggle() method is used to toggle between slideUp() and slideDown() method. If the element is slide down,it will slide up the element and if it is slideup,it will slide down.

Syntax:
---------

$(selector).slideToggle(); $(selector).slideToggle(speed); $(selector).slideToggle(speed,callback); $(selector).slideToggle(speed,easing,callback);

Parameters

|=====================================

Description

======================================|

Speed :


|======================================

it is an optional Parameters and specify the speed of the delay.it possible value are slow,fast,miliseconds.

======================================|

Easing :


|======================================

It specifies the easing function to be used for transition.

======================================|

Callback :


|======================================

It is an Optional parameter and specify the function to be called after comletion of slideToggle() effect.

======================================|
